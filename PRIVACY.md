# Next New Tab 隐私政策

**最后更新：2026-05-18**

## 概述

Next New Tab(以下简称"本扩展")是一款 Chrome 新标签页扩展程序,用于替换浏览器默认新标签页,提供书签快速访问、智能搜索栏和动态壁纸功能。

我们高度重视您的隐私。本政策说明本扩展如何处理您的数据。

## 我们收集的信息

**本扩展不收集、不存储、不上传任何用户的个人身份信息。**

本扩展仅在您的浏览器本地使用以下数据,所有数据均保存在您本机的 `chrome.storage` 中,绝不离开您的设备:

- 您的浏览器书签(通过 `chrome.bookmarks` API 读取,仅用于在新标签页展示与编辑)
- 您的扩展偏好设置(壁纸来源、当前壁纸缓存、搜索引擎选择、主题、布局等)
- 壁纸图片本地缓存

## 网络请求

为提供完整功能,本扩展会向以下公开服务发起 HTTP 请求,这些请求不包含任何用户身份数据:

- **您的书签所属域名**:仅用于读取该网站公开的图标资源(如 `/apple-touch-icon.png`、`/favicon-192x192.png` 等),以提升书签卡片视觉效果
- **www.bing.com**:获取 Bing 每日壁纸(`HPImageArchive.aspx` 接口)
- **wallpaper.apc.360.cn**:获取 360 壁纸库分类与图片列表
- **t1.gstatic.com**:作为图标兜底服务(Google Favicon API)
- **用户在设置中自定义添加的随机壁纸源 URL**

以上所有请求仅用于获取公开的图标和图片资源。

## 数据共享与转让

- 本扩展**不会**将任何用户数据出售给第三方
- 本扩展**不会**将任何用户数据转让给与扩展核心功能无关的目的
- 本扩展**不会**将任何用户数据用于确定信用度或贷款相关用途

## 权限说明

| 权限 | 用途 |
| ---- | ---- |
| `storage` | 在本地保存用户配置(壁纸、设置、缓存等) |
| `bookmarks` | 读取并编辑用户书签,以在新标签页中渲染 |
| `<all_urls>` 主机权限 | 获取书签网站图标与壁纸图片资源 |

## 远程代码

本扩展**未使用**任何远程代码(无远程 JS / Wasm 加载,无 `eval()`)。所有可执行代码均随扩展软件包一起发布并通过 Chrome 商店审核。
